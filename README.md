# 🖥️ 시스템 자원 AI 어드바이저 (System Resource AI Advisor)

본 프로그램은 노트북 및 PC의 시스템 자원(CPU, RAM, GPU, Disk, 전력 소모량 등)을 실시간으로 수집하고, OpenRouter LLM(AI 어드바이저)을 통해 시스템 상태를 종합적으로 진단받아 비정상적이거나 낭비되는 프로세스를 안전하게 최적화 및 제거할 수 있도록 도와주는 지능형 시스템 관리 프로그램입니다.

---

## 주요 기능

1. **실시간 하드웨어 모니터링**: CPU, RAM, GPU, 디스크 사용 상태 및 실시간 전력 사용량(Intel RAPL, NVIDIA, 노트북 배터리 등)을 실시간으로 추적 및 시각화합니다.
2. **AI 자원 분석 및 최적화 추천**: AI 엔진을 구동하여 낭비되는 백그라운드 프로세스나 파일 등을 분석하고 안전 기준에 맞는 최적화 조언을 제시합니다.
3. **일괄 조치 및 예외 관리**: AI가 추천하는 최적화 조치(`KILL_PROCESS`, `DELETE_FILE`)를 UI 상에서 일괄 승인/실행하거나 예외(화이트리스트)로 즉시 등록하여 편리하게 관리할 수 있습니다.
4. **네이티브 토스트 알림**: 자원 과점유 프로세스가 감지되면 작업 중에도 방해받지 않도록 Windows 알림 센터의 시스템 트레이 알림(토스트 알림)을 발송합니다.
5. **PyQt6 및 Streamlit 프론트엔드 제공**: 설치하여 바로 사용하는 데스크톱 GUI 프로그램(PyQt6)과 예쁘고 모던한 웹 프론트엔드(Streamlit) 두 가지 인터페이스를 함께 제공합니다.
